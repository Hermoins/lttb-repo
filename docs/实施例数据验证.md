## 2. 实施例算法实现

### 2.1 实施例一：基于一阶梯度（导数）的动态调整策略

#### 算法参数

| 参数   | 推荐值 | 说明                         |
| ------ | ------ | ---------------------------- |
| M      | 72     | 目标输出点数（压缩比约50:1） |
| B_base | 50     | 基础桶大小                   |
| α      | 1.0    | 梯度灵敏度因子               |
| B_min  | 5      | 桶大小下限（异常期高保真）   |
| B_max  | 100    | 桶大小上限（平稳期高压缩）   |
| Δt     | 1秒    | 采样时间间隔                 |

#### 核心公式

1. **梯度计算（中心差分）**:
```
G[i] = |C[i+1] - C[i-1]| / 2Δt
```

2. **动态桶大小**:
```
B_i = MAX(B_min, MIN(B_max, round(B_base / (1 + α × G_avg_i))))
```

3. **趋势预警斜率**:
```
S = Σ(P[j] - P̄)(j - j̄) / Σ(P[j] - P̄)²
```

### 2.2 实施例二：基于滑动方差的双阈值调整策略

#### 算法参数

| 参数    | 推荐值 | 说明                 |
| ------- | ------ | -------------------- |
| W       | 30     | 滑动窗口大小         |
| Th_low  | 0.02   | 低方差阈值           |
| Th_high | 0.2    | 高方差阈值           |
| B_min   | 5      | 桶大小下限           |
| B_max   | 100    | 桶大小上限           |
| β       | 0.5    | 指数加权移动平均系数 |

#### 核心公式

1. **滑动方差计算**:
```
Var[i] = (1/W) × Σ(C[j] - C̄)²
```

2. **三区域动态分桶**:
```
if Var_i < Th_low:    B_i = B_max           # 平稳区
elif Var_i <= Th_high: B_i = 线性插值        # 过渡区
else:                  B_i = B_min           # 异常区
```

3. **平滑处理**:
```
B_i_smooth = β × B_i + (1-β) × B_{i-1}_smooth
```

### 2.3 实施例三：基于多参数融合的自适应策略

#### 算法参数

| 参数 | 推荐值 | 说明     |
| ---- | ------ | -------- |
| w1   | 0.6    | 梯度权重 |
| w2   | 0.4    | 方差权重 |

#### 核心公式

1. **归一化**:
```
G_norm[i] = G[i] / max(G)
Var_norm[i] = Var[i] / max(Var)
```

2. **融合评分**:
```
R[i] = w1 × G_norm[i] + w2 × Var_norm[i]
```
