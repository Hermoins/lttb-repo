## 三、提交前的必做事项

### 3.1 新颖性检索（强烈建议）

在提交前，建议进行初步检索确认创新性：

**推荐检索方式**：
1. **自行检索**：使用免费数据库
   - 中国知网（CNKI）
   - 国家知识产权局专利检索系统
   - SooPat专利搜索

2. **委托检索**：付费服务
   - 专利局检索中心
   - 专业检索机构

**检索关键词**：
```
中文：动态LTTB、自适应采样、瓦斯预警、矿山安全监控
英文：Dynamic LTTB, Adaptive downsampling, Gas trend prediction
```

### 3.2 时间戳/优先权证明

| 证明方式   | 说明                 |
| ---------- | -------------------- |
| 发明日志   | 记录研发时间线       |
| 邮件记录   | 显示研发进展的邮件   |
| 内部文档   | 公司内部技术文档日期 |
| 云盘时间戳 | 证明文档创建时间     |

**建议**：保存好研发过程中的所有文档和沟通记录
